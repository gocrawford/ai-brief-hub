# YouTube Brief — Output Format Reference

Use these exact section templates when writing the weekly YouTube AI Intelligence Brief.

## Header

```markdown
# Weekly YouTube AI Intelligence Brief
**Date:** [Current Date]
**Prepared for:** Bill, AI & Innovation Leader

---
```

## Section 1: Executive Summary

```markdown
## 1. Executive Summary
```

Write 4-5 numbered key themes. Each theme: bold title + 2-3 sentence explanation.

```markdown
1. **[Theme Title]:** [2-3 sentence explanation of the theme and why it matters for Bill.]
2. **[Theme Title]:** [...]
```

## Section 2: Top Picks (Highest Scored Content)

```markdown
## 2. Top Picks (Highest Scored Content)
```

Feature the **top 3-5 highest-scoring videos** (minimum score: 8.5/10). Each pick:

```markdown
### [Video Title]
**Video:** [Full Video Title](URL) ([Channel Name]) [reference number]
**Score: [X.X]/10** ([dimension 1] [score], [dimension 2] [score], [dimension 3] [score])

**Summary:** [3-5 sentence summary of the video's key content, written in professional prose.]

**Key Takeaways for Bill:** [Paragraph summarizing the key takeaways from the video and their relevance to Bill's work.]

**Suggested Test:** [Paragraph with a specific, actionable recommendation tied to Bill's role and tool stack.]
```

## Section 3: Tool Updates

```markdown
## 3. Tool Updates
```

Table of tools that received updates or new features this week:

```markdown
| Tool | Update/Feature | Impact |
| :--- | :--- | :--- |
| **[Tool Name]** | [What changed] | [Why it matters for Bill's team. Include reference.] |
```

Minimum 4 rows. Prioritize tools in Bill's stack.

## Section 4: Tutorials & How-Tos

```markdown
## 4. Tutorials & How-Tos
```

Feature 2-3 practical tutorial videos:

```markdown
### [Tutorial Title]
**Source:** *[Video Title]* ([Channel Name]) [reference number]

**Practical Steps:**
1. **[Step Name]:** [Description]
2. **[Step Name]:** [Description]
```

## Section 5: Use Cases & Demos

```markdown
## 5. Use Cases & Demos
```

Feature 2-3 real-world use case or demonstration videos:

```markdown
### [Use Case Title]
**Source:** *[Video Title]* ([Channel Name]) [reference number]

**Use Case:** [One sentence describing the use case.]
**Application:** [2-3 sentences on how it works.]
**Action for Bill:** [Specific recommendation for Bill's team.]
```

## Section 6: Emerging Signals

```markdown
## 6. Emerging Signals
```

3-5 bullet points identifying early trends:

```markdown
* **[Signal Title]:** [2-3 sentence description of the emerging trend and why Bill should watch it. Include reference.]
```

## Section 7: Full Video Index

```markdown
## 7. Full Video Index
```

Comprehensive table of ALL videos reviewed (minimum 15), sorted by score descending:

```markdown
| Score | Title | Channel | Summary | Link |
| :--- | :--- | :--- | :--- | :--- |
| **[X.X]** | [Video Title] | [Channel] | [One-sentence summary.] | [URL] |
```

## Section 8: Suggested Search Refinements for Next Week

```markdown
## 8. Suggested Search Refinements for Next Week
```

3-5 bullet points:

```markdown
* **Add:** [New search term or channel to monitor, with rationale.]
* **Remove:** [Search term or channel that yielded low signal, with rationale.]
```

## References

```markdown
---
### References
[1] [URL] "[Video Title]"
[2] [URL] "[Video Title]"
```

Every video cited in the brief must have a numbered reference with a direct YouTube URL.
