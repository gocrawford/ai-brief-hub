# Social Brief — Output Format Reference

Use these exact section templates when writing the weekly Social Media AI Intelligence Brief.

## Header

```markdown
# Weekly Social Media AI Intelligence Brief
**Date:** [Date Range]
**Prepared for:** Bill (AI and Innovation Leader)
**Category:** Social — Practical AI Signals from Social Media

---
```

## Section 1: Executive Summary

```markdown
## 1. Executive Summary
```

Opening paragraph (2-3 sentences), followed by:

```markdown
**Top 3 Social Media AI Signals This Week:**
1. **[Signal Title]:** [2-3 sentence description and why it matters for Bill.]
2. **[Signal Title]:** [...]
3. **[Signal Title]:** [...]
```

## Section 2: Top Picks

```markdown
## 2. Top Picks
```

Top 3-5 highest-scoring posts across all platforms. Each pick uses a metadata table followed by prose:

```markdown
### [N]. [Content Title]

| Field | Detail |
| :--- | :--- |
| **Platform** | [TikTok/LinkedIn/Instagram/Facebook] |
| **Creator** | [Name] (@handle) |
| **URL** | [View on Platform](URL) |
| **Views** | [count] |
| **Likes** | [count] |
| **Posted** | [Date] |
| **Score** | [X.X]/10 |

[2-3 paragraph description of the content, its key message, community reaction, and significance.]

**Practical Takeaway for Bill's Team:** [2-3 sentences with specific, actionable recommendations.]
```

## Section 3: LinkedIn Highlights

```markdown
## 3. LinkedIn Highlights
```

3-5 notable LinkedIn posts, each as a subsection:

```markdown
### [Author Name] on [Topic]

[1-2 paragraph description of the post, key insight, and community reaction.]

**Engagement:** [Likes] Likes | [Comments] Comments | [Shares] Shares
**Link:** [View on LinkedIn](URL)
```

