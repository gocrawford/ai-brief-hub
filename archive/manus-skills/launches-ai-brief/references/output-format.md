# Launches Brief — Output Format Reference

Use these exact section templates when writing the weekly Launches AI Intelligence Brief.

## Header

```markdown
# Weekly Launches AI Intelligence Brief
**Date:** [Date Range]
**Category:** Launches — New AI Tools & Platforms Businesses Are Using

---
```

## Section 1: Executive Summary

```markdown
## 1. Executive Summary
```

Opening paragraph (2-3 sentences on the week's launch landscape), followed by:

```markdown
**Top 5 Launches Bill's Team Should Know About:**
1. **[Product Name]:** [1-2 sentence description of the launch and why it matters.]
2. **[Product Name]:** [...]
3. **[Product Name]:** [...]
4. **[Product Name]:** [...]
5. **[Product Name]:** [...]
```

## Section 2: Top Picks

```markdown
## 2. Top Picks
```

Top 3-5 highest-scoring launches:

```markdown
### [Product Name]
* **What it is:** [2-3 sentence description. Include reference number.]
* **Why it matters:** [1-2 sentences on how it compares to existing tools and why this is significant.]
* **Pricing:** [Pricing model and specific prices if available.]
* **Evaluation Recommendation:** **[Evaluate Now / Monitor / Add to Watchlist].** [1-2 sentences explaining the recommendation.]
```

## Section 3: Product Hunt Highlights

```markdown
## 3. Product Hunt Highlights
```

Table sorted by upvotes (minimum 8 rows):

```markdown
| Product | Upvotes | Description |
| :--- | :--- | :--- |
| **[Product Name]** | [count] | [One-sentence description.] |
```

Filter for AI/developer/productivity tools only.

## Section 4: GitHub Trending

```markdown
## 4. GitHub Trending
```

Table of top AI repositories (minimum 6 rows):

```markdown
| Repository | Stars | Description |
| :--- | :--- | :--- |
| **[owner/repo]** | [count] | [One-sentence description.] |
```

Focus on AI agents, frameworks, and tools.

## Section 5: YC & Startup Ecosystem

```markdown
## 5. YC & Startup Ecosystem
```

3-6 notable companies:

```markdown
* **[Company Name]:** [2-3 sentences: what they do, funding amount/valuation if applicable, and why they matter. Include reference number.]
```

## Section 6: Enterprise AI Platforms

```markdown
## 6. Enterprise AI Platforms
```

3-5 enterprise-grade launches/updates:

```markdown
* **[Platform Name]:** [2-3 sentences: what launched/updated, competitive positioning, and relevance to Bill's Microsoft-shop enterprise. Include reference number.]
```

## Section 7: Open Source & Developer Tools

```markdown
## 7. Open Source & Developer Tools
```

3-5 open-source/developer launches:

```markdown
* **[Tool Name]:** [2-3 sentences: what it does, how developers are using it, and relevance to Bill's stack. Include reference number.]
```

## Section 8: Launches to Watch

```markdown
## 8. Launches to Watch
```

3-5 promising early-stage tools:

```markdown
* **[Tool Name]:** [1-2 sentences on what it does and why it's worth watching.]
```

## Section 9: Full Launch Index

```markdown
## 9. Full Launch Index
```

Comprehensive table (minimum 20 rows), sorted by score descending:

```markdown
| Score | Product | Category | Source | Pricing | One-Line Summary |
| :--- | :--- | :--- | :--- | :--- | :--- |
| [X.X] | [Name] | [Category] | [Source] | [Price] | [One sentence.] |
```
